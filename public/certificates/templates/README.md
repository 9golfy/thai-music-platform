# Certificate Templates

วางไฟล์ template ใบประกาศ (JPG) ในโฟลเดอร์นี้

## ไฟล์ที่ต้องมี:

1. `default-template.jpg` - Template มาตรฐาน (1024x768px)
2. `gold-template.jpg` - Template สีทอง สำหรับโรงเรียน 100% (1024x768px)
3. `silver-template.jpg` - Template สีเงิน สำหรับโรงเรียนสนับสนุนฯ (1024x768px)

## ข้อกำหนด:

- ขนาด: 1024x768 pixels (4:3 ratio)
- รูปแบบ: JPG
- คุณภาพ: สูง (90-100%)
- พื้นที่สำหรับชื่อโรงเรียน: ตรงกลาง ประมาณ 45-48% จากด้านบน

## ตำแหน่งข้อความ (กำหนดใน lib/config/certificateTemplates.ts):

### ชื่อโรงเรียน:
- ตำแหน่ง: กึ่งกลาง (50% left, 45-48% top)
- ขนาดตัวอักษร: 42-48px
- สี: ดำ (#1a1a1a) หรือตามสี template
- จัดกึ่งกลาง

### เลขที่ใบประกาศ:
- ตำแหน่ง: ซ้ายล่าง (20-25% left, 75-78% top)
- ขนาดตัวอักษร: 14-16px

### วันที่ออก:
- ตำแหน่ง: ขวาล่าง (75-80% left, 75-78% top)
- ขนาดตัวอักษร: 14-16px

## วิธีปรับแต่งตำแหน่ง:

แก้ไขไฟล์ `lib/config/certificateTemplates.ts` เพื่อปรับตำแหน่งและสไตล์ของข้อความ
